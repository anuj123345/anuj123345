# 💫 About Me:
As a Management Information System Consultant at Odisha School Education Program Authority, I contribute to the World Bank-funded STARS project by managing and analyzing government data to drive educational reforms. My role focuses on maintaining data accuracy, creating dynamic Excel-based dashboards, and performing advanced data analysis to empower stakeholders with actionable insights.  <br><br>Holding a BTech in Electrical and Electronics Engineering from Siksha 'O' Anusandhan University, I bring strong technical expertise in data visualization tools like Tableau, as well as skills in Statsmodels and TensorFlow. My work is underscored by a commitment to utilizing data-driven methodologies for impactful decision-making in the education sector
